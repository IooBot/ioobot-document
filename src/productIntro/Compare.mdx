# 方案对比

|             | leancloud 其他类似 |                 我们                  |
| :---------- | :------------: | :--------------------------------------: |
| ** API **     |      REST      |                 Graphql                  |
| ** backend ** | leancloud SDK  |          主流云服务商的接口、open source           |
| ** DB **      |      不公开       | 开源 DB（已完成：mongoDB， 开发中：MySQL and others  ) |
| ** 适用阶段 **    |       开发       |               开发、测试、商用（生产）               |
| ** 计费方式 **    |       天        |            按需（秒或小时），不用时不需要付费             |
| ** 数据迁移 **    |    暂未发现     |                    自由                    |
| ** 扩展 **      |       收费       |                按需，不需要额外付费                |
