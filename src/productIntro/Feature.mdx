# 产品特色优势

## 技术优势

我们一直在做基于云服务器和云函数的开发，积累了丰富的经验和技术。

## 产品优势

 1. 把云函数和web 服务结合起来，建网站不再需要云服务器
 2. 基于云函数实现微信公众号认证等高级服务，实现微信公众号免服务器
 3. 自动生成数据库查询API，支持符合graphql标准的数据库查询
 4. 云函数的报错信息，直接发送到钉钉的消息频道中，方便监控

## 不适合的场景

1. 需要长时间数据在线的应用：视频、直播等

2. 长时间不间断运行的应用 ：大数据分析
